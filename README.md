# Online Bookstore Database System (SQL Project)

## Overview
A database system designed for an online bookstore to manage books, customers, orders, shipping, and reviews.

## Features
- Book inventory management  
- Customer orders and accounts  
- Shipping and delivery tracking  
- Reviews and wishlist system  

## Technologies
- SQL  
- ERD Design  
- Database Normalization (1NF, 2NF, 3NF)

## Key Work
- Designed ERD and database schema  
- Applied normalization techniques  
- Created tables and inserted data  
- Wrote SQL queries using JOIN and GROUP BY
